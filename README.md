# Projeto React com useState

Aplicacao simples desenvolvida para demonstrar o uso do `useState` na gestao de estado local em React.

## Funcionalidades

- contador com botoes para incrementar, decrementar e zerar
- formulario com campo de nome
- mensagem exibida na tela apos o envio do formulario

## Como executar

```bash
npm install
npm run dev
```

## Objetivo

Atender ao enunciado da atividade com um exemplo basico, claro e funcional usando estado local em um componente React.
